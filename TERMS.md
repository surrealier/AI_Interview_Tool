# Beta Terms

AI Interviewer v0.9.0-beta is a paid beta product for interview practice. It is
not a hiring decision system and must not be used as the sole basis for
employment, admission, promotion, or compensation decisions.

## Beta Scope

The beta supports one local Windows user, local session storage, external model
folders, Qwen3-TTS or Windows SAPI TTS, optional microphone recording, optional
STT, and optional follow-up question generation.

## Limitations

- Voice quality depends on installed models, GPU/CUDA, and local audio devices.
- STT accuracy is not guaranteed.
- Follow-up questions are practice prompts, not professional career advice.
- Unsigned beta builds may show Windows SmartScreen warnings.
- Code signing, auto-update, and enterprise administration are planned for a
  later release.

## User Responsibilities

Users are responsible for obtaining rights to model weights, videos, voices,
questions, and other content they add to the app. Users must not record or
process another person's voice without proper consent.

## Warranty

The beta is provided as is. Surrealier does not guarantee uninterrupted use,
particular interview outcomes, or compatibility with every Windows, GPU, driver,
or audio device configuration.
